# Ledger Instinct — T-Account Focus Drill

A focused accounting practice tool for developing debit-and-credit placement instincts through short transaction drills.

## Features

- Debit-versus-credit T-account decisions
- Immediate explanatory feedback
- Streak and best-score tracking
- Keyboard-friendly progression
- Responsive single-file implementation
- No external dependencies or build step

## Run locally

Open `index.html` in a modern browser.

## GitHub Pages

Pushes to `main` are deployed automatically by the GitHub Pages workflow. Once deployment completes, the site is available at:
